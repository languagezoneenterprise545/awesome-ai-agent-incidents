## 🔒 Safety and trust checks

Autonomous agents can act on their own once they have access to tools. That can create risk when they can read email, browse the web, change files, or call external services.

Use the corpus to check for issues like:

- Prompt injection from web pages, messages, or documents
- Unwanted tool calls
- Data leakage from private context
- Confused goals and bad task handoff
- Weak checks before the agent takes action
- Over-trusting model output
- Missing approval steps for sensitive work

If you manage an agent system, this material can help you build safer review rules and better guardrails.
